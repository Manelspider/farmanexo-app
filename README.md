# FarmaNexo App

## Descripción

Aplicación móvil para acceso a FarmaNexo. Permite a médicos y proveedores consultar recetas y estados desde cualquier lugar. Por el momento orientada a pruebas de despliegue y conectividad con la API.

## Características principales

- Consulta de recetas médicas y estado.
- Acceso rápido y seguro mediante autenticación.
- Integración con la API de FarmaNexo.
- Preparada para despliegue multiplataforma (iOS/Android).
